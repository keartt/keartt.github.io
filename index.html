간략 소개

3년차 Java 백엔드 개발자로, Spring Boot 기반 웹 서비스와 PostgreSQL/PostGIS 공간 데이터 처리, 지도 기반 GIS 서비스 개발을 주력으로 합니다. 세션 인증을 JWT로 전환하고 OAuth2·RBAC 기반 인증/인가 체계를 직접 설계하는 등 백엔드 아키텍처를 주도해 온 경험이 있습니다.

Java 백엔드를 중심으로 React 프론트엔드(Cesium·Three.js 3D/GIS)와 Python(FastAPI) 데이터·AI 연계 서버까지 서비스 전 계층을 직접 구현한 풀스택 역량을 갖추고 있으며, LangChain 기반 RAG 스마트검색 등 AI 연계 프로젝트도 수행했습니다.

기능 구현보다 유지보수성과 성능을 우선하며, 성능 문제는 추측이 아닌 실행계획(EXPLAIN)과 로그를 근거로 분석·개선하는 것을 원칙으로 합니다. 효율적인 협업과 확장 가능한 구조 설계를 중요하게 생각합니다.

Spring Boot 안전관리 백엔드 아키텍처 설계·구축
19개 REST 컨트롤러 / 26개 서비스 / 33개 엔티티(약 18.7k LOC) 규모의 백엔드를 설계. 감사 필드 엔티티는 JPA로, 복잡한 통계·동적 검색 쿼리는 MyBatis(XML)로 분리한 이중 영속성 하이브리드 구조로 생산성과 조회 성능을 동시에 확보.
세션 → JWT 인증 전환 및 통합 인증 허브 구축
RS256 서명 키를 PEM으로 로드해 secret 노출 없이 토큰을 발급·검증. Access Token은 httpOnly 쿠키, Refresh Token claims는 Redis에 저장하고, RT 회전 발급 + 사용자당 토큰 수를 Redis ZSet으로 추적해 초과 시 최고령 토큰을 자동 폐기. OAuth2 소셜 로그인(Google/Kakao/Naver)과 역할 기반 페이지 권한(RBAC)을 자체 구현하고, 다수 연계 프로젝트의 로그인을 대신 처리하는 인증 허브로 운영.
실시간 안전 알림 시스템 (SSE)
SseEmitter 기반 실시간 알림을 구축하고 Redis 캐시로 다중 클라이언트 알림 성능을 최적화. 점검 기한 기준으로 점검예정·기한도래·기한초과 상태를 자동 판정해 별도 조작 없이 알림 상태가 갱신되도록 구현.
공간정보(GIS) 처리 및 지오코딩
PostGIS + Hibernate Spatial/JTS로 공간 객체를 JPA에서 직접 영속화. proj4j로 TM 중부원점(EPSG:5179/5186)↔WGS84 좌표 변환을 구현하고, 주소→좌표 지오코딩과 지도 통합판 타일 생성을 스케줄러 배치로 자동화.
3D BIM 파이프라인 및 외부 시스템 연계
IFC 3D 모델 업로드 → 파싱 → GLB 변환 → 3D 타일 생성 파이프라인을 Spring 스케줄러 배치로 자동화. 도로명주소·VWorld·ITS CCTV·건축물대장·Google Analytics 등 이질적 외부 시스템을 공통 API 클라이언트로 추상화하고, 성균관대 AI 결함탐지 서버와는 REST + SSHJ(SFTP)로 결과 파일·이미지 수집 파이프라인을 구축.
풀스택 — React 3D 지도·BIM 뷰어 & Python AI 연계
React 19 + Vite + TailwindCSS 기반 3D 안전관리 플랫폼(약 49k LOC)을 주도 개발. Cesium(3D 지도)과 Three.js/web-ifc(BIM 뷰어)를 통합하고 라벨 디클러터링·POI 클러스터링 등 대용량 3D 렌더링 UX를 개선. Python/FastAPI로 IFC 변환 서버와 LangChain·Ollama·PGVector 기반 RAG 스마트검색 서버를 구축해 백엔드와 연계.
